# Example databases

## How to use

### Basic example

```js
fetch(
	'https://my-json-server.typicode.com/claumartinezh/training-db/shoes'
).then((response) => {
	//do something
});
```

This DB works with [JSON Server](https://github.com/typicode/json-server)
